О программе

Программа была создана по теме "Регистратура поликлиники" для курсового проекта студентом КМРК группы 19-ИС-13 Антипиным Владиславом.
Программа "RegPol" представляет собой Автоматизированную Информационную Систему для: создания отчётов(талонов), хранения талонов, работы с хранимыми данными в базе данных.  
Также она ускоряет поиск и создание талонов, что ускоряет работу поликлиники.
Программа создана в средах разработки Delphi10.3, Delphi10.4,Delphi11.1 на языке Object Pascal, реалезована возможность работать с Microsoft word© Для создания отчётов.

Минимальные системные требования: 4Гб. оперативной памяти, 2-х ядерный процессор, места на жёском диске 150Мб, доступ к локальной сети.
Операционная система: Win10X64,Win10X32,Win7X64,Win7X32.
