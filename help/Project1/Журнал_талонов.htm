Справочник талонов в программе АИС "RegPol"

Справочник талонов точно такой же как и другие справочники т.е. есть кнопки навигации изб.20

изб.20

Но все жё есть свои отличия. Есть кнопки для работы данными(Смотри изб.21).

изб.21

Для удаления данных необходимо выбрать одну из записей при помощи кнопок навигации, а далее жмём кнопку "Удалить".
Вторая кнопка необходим для создания нового талона.

Также есть кнопка вывода данных в Microsoft word. изб.22

 изб.22

Чтобы вывести данные в виде отчёта в Microsoft word, необходимо выбрать нужную запись при помощи кнопок навигации и нажать кнопку "Вывести талон" Талон будет лежать в папке с программой Doc/Талоны
