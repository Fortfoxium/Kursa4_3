Справочник врачей в программе АИС "RegPol"

Для входа в окно с услугами необходимо перейти Главное меню ->Справочники ->Доктори нажать один раз, как на изб.18


изб.18

В открывшемся окне мы наблюдаем стандартный интерфейс для справочников, с кнопками навигации и кнопками для работы с данными.

    
  изб.19

Для добавления данных необходимо использовать кнопку "Добавить". После нажатия на неё поля с данными очистятся, а поле с ID инкрементируется. Далее необходимо вписать данные в поля и нажать кнопку сохранить 
Для изменения данных необходимо использовать кнопку "Изменить". После нажатия на неё поля с данными разблокируются, после изменения нажимаем кнопку сохранить. После этого данные сохранятся.
Для удаления данных необходимо использовать кнопку "Удалить". После нажатия запись удалится.
Если необходимо найти конкретную запись, есть поле поиска, там написано что необходимо писать для поиска записи.
