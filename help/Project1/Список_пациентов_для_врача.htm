Список пациентов записанных по данной услуги

Для открытия формы с созданием списка пациентов и его вывода в Excel microsoft, необходимо в основной форме открыть главное меню->Отчёты->список пациентов для врача
Откроется форма (изб.26) при выборе дня и услуги, если кто-то есть записанный в этот день, то кнопка разблокируется, иначе заблокируется
